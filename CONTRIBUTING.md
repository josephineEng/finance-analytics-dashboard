# Contributing to Finance Analytics Dashboard

Thank you for your interest in contributing! This project welcomes contributions from the community.

## How to Contribute

### 1. Fork the Repository

Click the "Fork" button on GitHub to create your own copy.

### 2. Clone Your Fork

```bash
git clone https://github.com/yourusername/finance-analytics-dashboard.git
cd finance-analytics-dashboard
```

### 3. Create a Feature Branch

```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes

- Follow the coding style guidelines
- Add comments for complex logic
- Update documentation as needed

### 5. Test Your Changes

```bash
# Run the dashboard
streamlit run dashboard.py

# Test database functions
python src/database.py
```

### 6. Commit Your Changes

```bash
git commit -m "Add: description of your changes"
```

### 7. Push to Your Fork

```bash
git push origin feature/your-feature-name
```

### 8. Create a Pull Request

- Go to the original repository
- Click "New Pull Request"
- Select your branch and describe your changes

## Coding Guidelines

- Use PEP 8 style guide for Python
- Add docstrings to functions
- Use meaningful variable names
- Comment complex logic
- Keep functions focused and modular

## Commit Message Convention

```
Type: Brief description

- More detailed explanation if needed
- Additional context or rationale
```

Types:

- `Add`: New feature
- `Fix`: Bug fix
- `Refactor`: Code refactoring
- `Docs`: Documentation updates
- `Style`: Formatting/style changes
